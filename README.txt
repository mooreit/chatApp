How to run the chatApp:

- Programming language: python 3.0

- Install requirement package: 
	pip install simple-crypt

- In the terminal run:
 	To run server:	$python3 server.py 
	To run client:	$python3 client.py localhost 
	private chat, type as following: message + $$ + ID
	type 'exit' to exit the chat room.
